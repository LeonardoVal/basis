base This library is a bundle of assorted utility definitions used in our Javascript projects.
base.global A reference to the global object, which depends on the running environment. This is equal to process while running on NodeJS, to window while running on the browser's rendering thread, and to self if running on a web worker.
base.raise(message...)
base.raiseIf(cond, message...)
base.callStack(error)		 Functions raise() and raiseIf() are helper to raise errors. The function callStack() standardizes the way to obtain a call stack from an error, or from the place it is called.
base.obj(key, value...)
base.copy(objTo, objFrom...)		 Function obj() is a shortcut to make objects. Useful since Javascript's object literal notation does not allow expressions for the keys, as of ECMAScript 5. Function copy() is similar to Object.assign().
new base.Iterable(source)
base.iterable(sequence)		 Iterables (like in Java) are representations of sequences of values. An iterable can be built based on an array, a string, an object (key-value pairs) or an iterator function. An iterable can be converted to an array using toArray(), to a string using join(sep) or to an object using toObject().
The Iterable class has many utility functions to deal with sequences, including: map, filter, sum, foldl, zip, forEach, product, combinations and many more.

base.Randomness.MersenneTwister		 Randomness is the base class for pseudorandom number generation algorithms and related functions. A limitation with Javascript's Math.random function is that it cannot be seeded. This hinders its use for simulations and simular purposes.
